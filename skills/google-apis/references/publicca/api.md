### `publicca.projects.locations.externalAccountKeys.create`

**POST** `v1/{+parent}/externalAccountKeys`

Creates a new ExternalAccountKey bound to the project.

**Parameters:**

| Parameter | Type | In | Required | Description |
|-----------|------|-----|----------|-------------|
| `parent` | `string` | path | Yes | Required. The parent resource where this external_account_key will be created. Format: projects/[project_id]/location... |

**Request body**: `ExternalAccountKey`

**Response**: `ExternalAccountKey`

```typescript
const res = await publicca.externalAccountKeys.create({
  // parameters
});
```

**Required scopes:**
- `https://www.googleapis.com/auth/cloud-platform`
