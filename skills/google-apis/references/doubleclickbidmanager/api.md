# DoubleClick Bid Manager API - API Reference

**Version**: `v2` | **Methods**: 7 | **Schemas**: 15

## Methods

| Method | HTTP | Path | Description |
|--------|------|------|-------------|
| `doubleclickbidmanager.queries.create` | POST | `queries` | Creates a new query. |
| `doubleclickbidmanager.queries.delete` | DELETE | `queries/{queryId}` | Deletes an existing query as well as its generated reports. |
| `doubleclickbidmanager.queries.get` | GET | `queries/{queryId}` | Retrieves a query. |
| `doubleclickbidmanager.queries.list` | GET | `queries` | Lists queries created by the current user. |
| `doubleclickbidmanager.queries.run` | POST | `queries/{queryId}:run` | Runs an existing query to generate a report. |
| `doubleclickbidmanager.queries.reports.list` | GET | `queries/{queryId}/reports` | Lists reports generated by the provided query. |
| `doubleclickbidmanager.queries.reports.get` | GET | `queries/{queryId}/reports/{reportId}` | Retrieves a report. |

### `Date`

Represents a whole or partial calendar date, such as a birthday. The time of day and time zone are either specified elsewhere or are insignificant. The date is relative to the Gregorian Calendar. This can represent one of the following: * A full date, with non-zero year, month, and day values. * A month and day, with a zero year (for example, an anniversary). * A year on its own, with a zero month and a zero day. * A year and month, with a zero day (for example, a credit card expiration date). Related types: * google.type.TimeOfDay * google.type.DateTime * google.protobuf.Timestamp
