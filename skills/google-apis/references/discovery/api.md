# API Discovery Service - API Reference

**Version**: `v1` | **Methods**: 2 | **Schemas**: 5

## Methods

| Method | HTTP | Path | Description |
|--------|------|------|-------------|
| `discovery.apis.getRest` | GET | `apis/{api}/{version}/rest` | Retrieve the description of a particular version of an api. |
| `discovery.apis.list` | GET | `apis` | Retrieve the list of APIs supported at this endpoint. |

### `discovery.apis.getRest`

**GET** `apis/{api}/{version}/rest`

Retrieve the description of a particular version of an api.

**Parameters:**

| Parameter | Type | In | Required | Description |
|-----------|------|-----|----------|-------------|
| `api` | `string` | path | Yes | The name of the API. |
| `version` | `string` | path | Yes | The version of the API. |

**Response**: `RestDescription`

```typescript
const res = await discovery.apis.getRest({
  // parameters
});
```

---

### `discovery.apis.list`

**GET** `apis`

Retrieve the list of APIs supported at this endpoint.

**Parameters:**

| Parameter | Type | In | Required | Description |
|-----------|------|-----|----------|-------------|
| `name` | `string` | query | No | Only include APIs with the given name. |
| `preferred` | `boolean` | query | No | Return only the preferred version of an API. |

**Response**: `DirectoryList`

```typescript
const res = await discovery.apis.list({
  // parameters
});
```
